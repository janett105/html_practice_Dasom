html 태그 5개

  1. input태그 : data를 삽입할 공간을 명시한다. type속성에 따라 삽입하는 정보의 종류를 달리 할 수 있다.
  2. div태그 : 정보를 구분하는 container를 만든다.줄바꿈.CSS와 함께 사용한다.
  3. script태그 : client-side script(ex:Java Script)를 사용한다.
  4. link태그 : 해당 문서와 external resource사이의 관계를 정의한다. 속성만이 존재한다. head내부에만 속할 수 있다.
  5. span태그 : 정보를 구분하는 inline container를 만든다.CSS와 함께 사용한다.

클라이언트와 서버의 의미

클라이언트 : 정보를 요구하는 곳
서버 : 정보를 제공하는 곳
